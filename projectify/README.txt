== Description ==

This plugin adds extra fields for users to add extra fields based on time, difficulty and cost. 
They can also make it a certain background color via the admin menu page.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `projectify` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

How do I add Projectify Info to my post?

The fields should be automatically rendered on the edit post page, right under your content. It should then appear above your post's content.

How do I change the color of my post of Projectify Info?

This can be done via the admin menu. Click the Projectify Link and there is an input where you can change the background color.
This field accepts color or hex code.
